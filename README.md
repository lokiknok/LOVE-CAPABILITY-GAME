# LOVE-CAPABILITY-GAME
Love Compatibility Calculator  A fun and interactive Love Compatibility Calculator built using HTML, CSS, and JavaScript. Enter two names and discover a compatibility score, relationship status, and an automatically selected romantic gift.

🎮 How It Works
Enter your name.
Enter your partner's name.
Click Calculate Compatibility.
The application generates:
A compatibility percentage
A relationship status
A personalized romantic message
An automatically selected gift

The same pair of names always produces the same compatibility result using a deterministic scoring algorithm.
